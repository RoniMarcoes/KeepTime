Introduction

Features:

  • Set an alarm at any time, in both 12- and 24-hour formats
  • Set a timer for any amount of time up to 365 days
  • Track time with the stopwatch
  • Use month/day/year format or day/month/year format
  • Include a message and sound with your alarm and timer
  • Repeat the alarm any day of the week
  • Repeat the timer indefinitely
  • Minimize the program to the system tray

Alarm

To use the alarm, use the three boxes to set the desired time. Include your message in the message box and check any repeat boxes if you want to repeat the alarm any other day.

Click the Start button when you want to start the alarm.

Timer

To use the timer, either use the up and down arrow buttons or enter your own numbers in the three boxes. Check the repeat box if you want to repeat the timer at the same interval indefinitely. Enter a custom message in the message box.

Click the Start button when you want to start the timer.

Stopwatch

Click the Start button when you want to start the stopwatch. There may be some lag before the stopwatch begins. When you want to stop the stopwatch, click the Stop button.

Clicking the Start button again will resume from your previous time. If you want to reset your time, just click the Reset button.
